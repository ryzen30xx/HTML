Floats

Floating an element removes it from the usual flow of the page structure, this can be very powerful but also damaging - we don't tend to use floats as often now as we have access to the more powerful flex and grid systems.

You can learn more about them at mdn web docs and mdn web docs - legacy layouts

Table

Tables are powerful when used to display tabular data, in legacy periods tables were used to layout whole websites and are still very common place in HTML emails. Unlike the previous layout systems, table layouts are created entirely in HTML and not CSS - however we can apply it via CSS to mimic the HTML markup if we wish.

Tables are great, but with limitations, we tend to use grids or other layouts now, due to it's legacy use as a layout system it is no longer covered by mdn web docs.
